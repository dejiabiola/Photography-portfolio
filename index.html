HTML Structure

The template is based on Bootstrap Framework - http://getbootstrap.com/
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.
 
Create a .row and add the appropriate number of .col-* columns.
  
<div class="row">
        
       { Level 1 column }

        <div class="row">
            <div class="col-6"> { Level 2 } </div>
            <div class="col-6"> { Level 2 } </div>
        </div>

</div>

If you need more information, please visit this site: http://getbootstrap.com/css/#grid

Variations

Apart from the main template, there are four variations in which you can choose from.

These are:

  • Video hero section: In this variation you can have a video play in the background of the hero section.
  • Sliding image hero section: In this variation, there is a carousel of images with a fading transition between them every two seconds
  • Sticky nav: In this variation, the navigation is sticks to the top of the viewport when the user scrolls down.
  • Gif nav: In this variation, there is a video playing in the background of the hero section, when a user starts scrolling down, the navigation bar sticks to the top of the viewport and a gif appears as the background of the navigation bar.

How to Edit the Variations

Video Hero Variation

The main code for this variation is the video element located in the header element in index.html. You can easily edit it to any video of your choice by changing the src attribute to the link of any videp of your choice

            <video loop muted autoplay preload="auto">
            <source src="https://res.cloudinary.com/dejiabiola/video/upload/v1595611393/Time_Lapse_Video_Of_Night_Sky.mp4" type="video/mp4">
          Your browser does not support video
        <video>

Gif Navbar Variation

This variation sets the bacground of the navbar with a gif when a user starts scrolling. The gif can be custimised to your preference To change the gif, you can replace the current one with another of your choice in the custom.css file

            .navbar.scroll-down {
              background: url("../gif/ezgif.com-optimize.gif");
              background-repeat: no-repeat;
              background-size: cover;
              object-fit: cover;
            }

Fonts

To change the embedded font, please take a look in the head part of assets/css/custom.css:
  
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@500;600;700&family=Poppins:wght@300;400;500;600;700&display=swap');

 

To change the font, first go to http://www.google.com/webfonts choose a font and add the generated code to the top of global.css.
 
Then edit --header-font or --text-font in
  
:root {
  --header-font: 'Playfair Display', serif;
  --text-font: 'Poppins', sans-serif;
  --primary-color: #202020;
  --secondary-color: #FFFFFF;
  --grey-color: #777777;
  --button-hover-color: rgb(49, 47, 47);
  --black-color: #000000;
  --footer-links-color: #CDCDCD;
}

Sliders

Slick slider is the slider you will find in the content. It's an essay to use lighweight carousel with some decent features.
 
Gallery section slider has class 'slider-gallery' and each slide has '<div class="slide-wrapper-gallery">' wrapper for easy find where slide html code start.
My work section slider has class 'slider-mobile' and each slide has '<div class="slide-wrapper">' wrapper. If you wont to remove some slides just remove div with all content like this '<div class="item">'
  
<!-- Sliders -->
<section class="sliders">
      <div class="slide-wrapper">
          <img src="./assets/images/my-work-image.png" alt="Wedding Image" class="img-fluid slider-img" >
          <div class="slider-desc p-2 px-3">
            <p class="slider-name m-0">Mr & Mrs Ruth</p>
            <a href="#" class="slider-link">June 22, 2020</a>
          </div>          
      </div>

      <div class="slide-wrapper">
            ...
      </div>

      <div class="slide-wrapper">
            ...
      </div>
</section>
 
If you want change option for sliders you need to open assets/js/custom.js and find '// slider mobile' function and set true/false for option what you want to change
 
  
// Slick Slider Configuration
$(document).ready(function () {
  $('.slider-mobile').slick({
    slidesToShow: 3,
    cssEase: 'linear',
    autoplay: true,
    autoplaySpeed: 3500,
    responsive: [
      {
        breakpoint: 1068,
        settings: {
          arrows: true,
          slidesToShow: 3,
        },
      },
      {
        breakpoint: 768,
        settings: {
          arrows: true,
          slidesToShow: 2,
        },
      },
      {
        breakpoint: 480,
        settings: {
          slidesToShow: 1,
          slideToScroll: 1,
          centerMode: true,
          centerPadding: '30px',
          autoplay: true,
          autoplaySpeed: 2000,
        },
      },
    ],
  })
});

You can find more documentation here https://kenwheeler.github.io/slick/
